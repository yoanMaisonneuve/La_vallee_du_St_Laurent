# La Vallée du Saint-Laurent — Institut St-Laurent

Dépôt de travail du **Think Tank ISL** — souveraineté AGI, robotique humanoïde et industrialisation du corridor Saint-Laurent.

## Structure

```
├── Institut St-Laurent · Direction stratégique AGI-1.html   ← Site web Blueprint (prototype immersif)
├── three-scenes.js                                           ← Moteur WebGL Three.js du site
└── Institut St-Laurent Think Tank/
    ├── rapports/          Rapports de recherche AGI, fractal de Terman, rapport unifié, résumé exécutif
    ├── robotique/         Axe robotique humanoïde, pivot industriel Rive-Nord
    ├── feuille-de-route/  Feuille de route d'exécution stratégique
    ├── manifeste/         Manifeste ISL et document de positionnement
    └── media/             Vidéo pivot robotique, captures, images
```

## Site web

Ouvrir `Institut St-Laurent · Direction stratégique AGI-1.html` directement dans un navigateur (nécessite `three-scenes.js` dans le même dossier).

Le site requiert une connexion internet pour charger Three.js via `esm.sh` et les polices Google Fonts.
